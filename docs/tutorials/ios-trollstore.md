# TrollStore 使用教程

## 什么是 TrollStore？

TrollStore 是一个利用 **CoreTrust 漏洞** 实现永久签名的工具。通过它安装的应用不会过期，无需重新签名。

## 支持范围

- **iOS 14.0 - 16.6.1**（部分版本需结合越狱）
- 安装方法因 iOS 版本不同而异

## 安装 TrollStore

### 方法一：TrollInstallerX（推荐）

1. 下载 [TrollInstallerX](https://github.com/alfiecg24/TrollInstallerX/releases)
2. 通过 AltStore / Sideloadly 签名安装到设备
3. 打开 TrollInstallerX，点击 Install TrollStore
4. 桌面出现 TrollStore 图标即成功

### 方法二：TrollHelperOTA

1. 在 Safari 中访问安装页面
2. 点击 Install TrollStore
3. 等待安装完成

::: warning 注意
安装方式因设备和 iOS 版本不同而有差异，请查阅最新信息确认适用方案。
:::

## 使用 TrollStore 安装应用

1. 下载 `.ipa` 或 `.tipa` 文件
2. 点击分享按钮 → 选择「用 TrollStore 打开」
3. 点击 Install 安装
4. 桌面出现应用图标，**永久有效**

## 推荐安装的应用

| 应用 | 功能 |
|------|------|
| **Filza** | 文件管理器 |
| **AppStore++** | 应用降级 |
| **CocoaTop** | 进程管理器 |
| **Santander** | 系统信息查看 |

::: tip 提示
TrollStore 安装的应用具有与系统应用相同的权限，可以访问更多系统功能。
:::
