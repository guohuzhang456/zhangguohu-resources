## 安装方法

1. 打开 Magisk → 模块 → 从本地安装
2. 选择下载好的 `.zip` 模块文件
3. 安装完成后重启手机

::: tip 提示
模块建议从 [官方仓库](https://github.com/Magisk-Modules-Repo) 或酷安下载，避免恶意模块。
:::

## 模块冲突排查

如果安装某模块后出现问题：

1. 进入安全模式（开机时按住音量键）
2. Magisk 会自动禁用所有模块
3. 逐个启用模块，确定是哪个导致的问题
