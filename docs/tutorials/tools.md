# 搞机工具箱推荐

## 电脑端工具

### Android 调试

| 工具 | 平台 | 用途 |
|------|------|------|
| **[Platform Tools](https://developer.android.com/tools/releases/platform-tools)** | Win/Mac/Linux | ADB & Fastboot 官方工具 |
| **[搞机助手](https://github.com/GojiAssistant/GojiAssistant)** | Windows | 图形化刷机工具 |
| **[SDK Platform Tools](https://developer.android.com/studio)** | Win/Mac/Linux | Android 开发工具包 |

### iOS 工具

| 工具 | 平台 | 用途 |
|------|------|------|
| **[AltServer](https://altstore.io/)** | Win/Mac | AltStore 服务端 |
| **[Sideloadly](https://sideloadly.io/)** | Win/Mac | IPA 签名工具 |
| **[3uTools](http://www.3u.com/)** | Windows | 刷机/越狱辅助 |
| **[iMazing](https://imazing.com/)** | Win/Mac | iOS 设备管理 |
