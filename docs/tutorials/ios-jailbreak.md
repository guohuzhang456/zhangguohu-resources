# iOS 越狱入门指南

## 什么是越狱？

越狱（Jailbreak）是获取 iOS 系统最高权限的过程。越狱后可以：
- 安装第三方应用（非 App Store）
- 使用 Cydia / Sileo 安装插件
- 修改系统主题和界面
- 文件系统自由访问

## 越狱前必读

::: warning 重要提醒
- 越狱可能导致设备不稳定、耗电增加
- 部分应用（银行、游戏）会检测越狱并拒绝运行
- 越狱后官方保修可能失效
- **建议使用备用机越狱**
:::

## Dopamine 越狱教程

### 步骤一：安装 Dopamine

1. 使用 [TrollStore](./ios-trollstore.md) 安装 Dopamine
2. 或者使用 [AltStore](https://altstore.io/) / Sideloadly 自签安装

### 步骤二：执行越狱

1. 打开 Dopamine 应用
2. 点击「越狱」按钮
3. 等待设备重启
4. 桌面出现 **Sileo** 图标即越狱成功

### 步骤三：必装插件

1. **ElleKit** — 插件注入框架
2. **iCleaner Pro** — 系统清理工具
3. **Filza** — 文件管理器

::: tip 提示
越狱环境崩溃后需要重新越狱：重新打开 Dopamine → 点击越狱即可恢复插件。
:::

## 越狱后注意事项

- 不要随意卸载 Cydia / Sileo 自带的基础包
- 定期备份插件列表
- 谨慎添加来源不明的源
